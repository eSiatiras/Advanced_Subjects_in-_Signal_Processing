Project 3 Requirements

=> Requirements for Exercise_1 and Exercise_2 are covered from Exercise_3.

=> Exercise_3 
	*Matlab R2017a (May work for different versions as well) has been linked to python interpreter and matlab engine has been imported to the python script 'main.py'
	*The Python Version Supported by the matlab core compiler is Python 3.5 and the linking can be achieved as per the following link https://www.mathworks.com/help/matlab/matlab_external/install-the-matlab-engine-for-python.html
	The additional libraries needed are can be installed with the following commands:
	 - pip install matplotlib==3.0.3
	 - pip install numpy==1.18.5
	 - pip install scipy==1.3.3
